# MrsJone 能量衡算工具 v4.9
> Aspen Plus 自动化数据提取与能量衡算报告生成工具

基于 Python + Tkinter 开发，通过 COM 接口直连 Aspen Plus，一键提取模块进出料流股数据、热负荷、轴功率，自动生成标准化 Word 汇总报告与 Excel 明细文件，完成能量平衡校验，大幅提升化工流程数据处理效率。

---

## 功能特性
- 直连 Aspen Plus，自动读取 `.bkp` 流程文件并运行模拟刷新数据
- 支持多模块批量提取，可按住 `Ctrl` 多选流程单元
- 自定义导出字段：流股名称、温度、压力、汽相分率、流量、焓值等
- 双格式输出：Word 汇总报告 + 单模块独立 Excel
- 实时数据预览面板，可视化查看所有流股信息
- 自动计算进料总焓、出料总焓、热负荷、轴功率及能量平衡误差
- 支持换热器、精馏塔、泵、压缩机、闪蒸、反应器等主流模块
- 可视化 GUI：状态指示灯、进度条、运行日志、一键打开输出目录

## 运行环境
### 系统与软件
- Windows 系统（依赖 Aspen COM 接口，仅支持 Windows）
- Aspen Plus V8.4 及以上版本
- Python 3.8 ~ 3.11

### 依赖安装
```bash
pip install python-docx pandas pywin32
```

## 使用步骤
1. **准备流程文件**
   将 Aspen Plus 流程保存为 `.bkp` 备份文件，路径不含中文、空格与特殊字符。

2. **启动程序**
   运行 Python 脚本，打开 GUI 界面。

3. **选择文件与目录**
   - 选择 Aspen `.bkp` 文件
   - 选择报告输出文件夹

4. **连接 Aspen**
   点击「连接 Aspen (读取模块)」，等待状态灯变绿即连接成功。

5. **选择模块与导出项**
   - 多选需要提取的模块
   - 勾选生成 Word / Excel
   - 勾选需要导出的流股字段

6. **开始提取**
   点击「开始提取选中模块」，等待任务完成。
   提取完成后可在预览面板查看数据，并一键打开输出目录。

7. **断开连接**
   使用完毕后点击「断开 Aspen」，关闭程序。

## 支持模块类型
- `HEATX` 换热器
- `RADFRAC` 精馏塔
- `PUMP` 泵
- `COMPR / MCOMPR` 压缩机
- `FLASH2 / FLASH3` 闪蒸罐
- `HEATER` 加热器
- `RPLUG / RSTOIC` 反应器

## 输出文件说明
### Word 汇总报告
`能量衡算报告_时间戳.docx`
- 统一封面与时间信息
- 各模块独立分页：负荷数据、流股明细、能量平衡验证
- 格式规范，可直接用于课程设计/毕业设计

### Excel 明细文件（按模块生成）
`模块名-能量衡算.xlsx`
- 表1：负荷表（W、Q）
- 表2：流股明细（自定义字段）
- 表3：能量平衡验证（Hin、Hout、误差）

## 常见问题
1. **连接 Aspen 失败**
   - 关闭已打开的 Aspen 重新连接
   - 确保 `.bkp` 路径纯英文无特殊字符
   - 以管理员身份运行脚本

2. **提取数据全为 0**
   - 等待 Aspen 模拟运行收敛完成
   - 检查流程连接与物性方法设置

3. **缺少依赖库**
   - 按提示执行 `pip install python-docx` 等安装命令
